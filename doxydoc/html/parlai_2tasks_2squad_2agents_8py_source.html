213 class SentenceTeacher(IndexTeacher):
214  """Teacher where the label(s) are the sentences that contain the true
215  answer.
216 
217  Some punctuation may be removed from the context and the answer for
218  tokenization purposes.
219

221  following format:
222  {
223  'context': <context>,
224  'text': <question>,
225  'labels': <sentences containing the true answer>,
226  'label_candidates': <all sentences in the context>,
227  'episode_done': True,

231  no separate context field.
