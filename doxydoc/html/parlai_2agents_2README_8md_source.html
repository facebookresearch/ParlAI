1 # Agents
2 
3 This directory contains a variety of different agents which use ParlAI's interface.
4 
5 ## Utility
6 
7 - **local_human**: receives human input from the terminal. used for interactive mode, e.g. `parlai/scripts/interactive.py`.
8 - **legacy_agents**: contains deprecated agent code for posterity
9 - **random_candidate**: returns a random candidate, if candidates are available. simple baseline agent.
10 - **remote_agent**: uses ZMQ to communicate with a different process, either on the local machine or on a remote host.
11 - **repeat_label**: sends back the label if available. good for sanity checks such as checking statistics of the base dataset.
12 - **repeat_query**: repeats whatever is said to it. simple baseline agent.
13 
14 ## Non-neural agents
15 
16 - **ir_baseline**: chooses response based on simple word overlap
17 - **retriever_reader**: used primarily for OpenSquad evaluation. retrieves documents from database and reads them back
18 - **tfidf_retriever**: returns candidate responses based on tfidf overlap
19 - **unigram**: returns top unigrams

21 ## Text-based neural networks
22 
23 - **bert_ranker**: BERT-based bi-ranker and cross-ranker retrieval models
24 - **drqa**: context-based question answering system
25 - **fairseq**: provides access to models from FAIR's FairSeq library (github.com/facebookresearch/fairseq)
26 - **language_model**: simple RNN-based language model
27 - **memnn**: memory network
28 - **ibm_seq2seq**: IBM's RNN-based sequence to sequence model
29 - **seq2seq**: our RNN-based sequence to sequence model
30 - **starspace**: embedding model
31 - **transformer**: both generative and retrieval-based transformer models
