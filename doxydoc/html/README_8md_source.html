6 
7 [ParlAI](http://parl.ai) (pronounced “par-lay”) is a python framework for
8 sharing, training and testing dialogue models, from open-domain chitchat to
9 VQA (Visual Question Answering).
10 
11 Its goal is to provide researchers:
12 
13 - **70+ popular datasets available all in one place, with the same API**, among them [SQuAD](https://rajpurkar.github.io/SQuAD-explorer/), [MS MARCO](http://www.msmarco.org/), [QuAC](https://www.aclweb.org/anthology/D18-1241), [HotpotQA](https://hotpotqa.github.io/), [QACNN & QADailyMail](https://arxiv.org/abs/1506.03340), [CBT](https://arxiv.org/abs/1511.02301), [BookTest](https://arxiv.org/abs/1610.00956), [bAbI Dialogue tasks](https://arxiv.org/abs/1605.07683), [Ubuntu Dialogue](https://arxiv.org/abs/1506.08909), [PersonaChat](https://arxiv.org/abs/1801.07243), [OpenSubtitles](http://opus.lingfil.uu.se/OpenSubtitles.php), [Wizard of Wikipedia](https://openreview.net/forum?id=r1l73iRqKm), [VQA-COCO2014](http://visualqa.org/), [VisDial](https://arxiv.org/abs/1611.08669) and [CLEVR](http://cs.stanford.edu/people/jcjohns/clevr/). See the complete list [here](https://github.com/facebookresearch/ParlAI/blob/master/parlai/tasks/task_list.py)
14 - a wide set of **reference models** -- from retrieval baselines to transformers.
15 - a large zoo of **pretrained models** ready to use off-the-shelf
16 - seamless **integration of [Amazon Mechanical Turk](https://www.mturk.com/mturk/welcome)** for data collection and human evaluation
17 - **integration with [Facebook Messenger](http://www.parl.ai/docs/tutorial_messenger.html)** to connect agents with humans in a chat interface
18 - a large range of **helpers to create your own agents** and train on several tasks with **multitasking**
19 - **multimodality**, some tasks use text and images
20 
21 ParlAI is described in the following paper:
22 [“ParlAI: A Dialog Research Software Platform", arXiv:1705.06476](https://arxiv.org/abs/1705.06476).
23 
24 See the [news page](https://github.com/facebookresearch/ParlAI/blob/master/NEWS.md) for the latest additions & updates, and the website [http://parl.ai](http://parl.ai) for further docs.

26 ## Installing ParlAI
27 
28 ParlAI currently requires Python3 and [Pytorch](https://pytorch.org) 1.1 or
29 newer. Dependencies of the core modules are listed in `requirement.txt`. Some
30 models included (in `parlai/agents`) have additional requirements.
31 
32 Run the following commands to clone the repository and install ParlAI:
33 
34 ```bash
35 git clone https://github.com/facebookresearch/ParlAI.git ~/ParlAI
36 cd ~/ParlAI; python setup.py develop
37 ```
38 
39 This will link the cloned directory to your site-packages.
40 
41 This is the recommended installation procedure, as it provides ready access to the examples and allows you to modify anything you might need. This is especially useful if you if you want to submit another task to the repository.
42 
43 All needed data will be downloaded to `~/ParlAI/data`, and any non-data files if requested will be downloaded to `~/ParlAI/downloads`. If you need to clear out the space used by these files, you can safely delete these directories and any files needed will be downloaded again.

58 ## Examples
59 
60 A large set of examples can be found in [this directory](./examples). Here are a few of them.
61 Note: If any of these examples fail, check the [requirements section](#requirements) to see if you have missed something.
62 
63 Display 10 random examples from the SQuAD task
64 ```bash
65 python examples/display_data.py -t squad
66 ```
67 
68 Evaluate an IR baseline model on the validation set of the Personachat task:
69 ```bash
70 python examples/eval_model.py -m ir_baseline -t personachat -dt valid
71 ```
72 
73 Train a single layer transformer on personachat (requires pytorch and torchtext).
74 Detail: embedding size 300, 4 attention heads, 2 epochs using batchsize 64, word vectors are initialized with fasttext and the other elements of the batch are used as negative during training.
75 ```bash
76 python examples/train_model.py -t personachat -m transformer/ranker -mf /tmp/model_tr6 --n-layers 1 --embedding-size 300 --ffn-size 600 --n-heads 4 --num-epochs 2 -veps 0.25 -bs 64 -lr 0.001 --dropout 0.1 --embedding-type fasttext_cc --candidates batch
77 ```

81 ## Code Organization
82 
83 The code is set up into several main directories:
84 
85 - [**core**](./parlai/core): contains the primary code for the framework
86 - [**agents**](./parlai/agents): contains agents which can interact with the different tasks (e.g. machine learning models)
87 - [**examples**](./parlai/examples): contains a few basic examples of different loops (building dictionary, train/eval, displaying data)
88 - [**tasks**](./parlai/tasks): contains code for the different tasks available from within ParlAI
89 - [**mturk**](./parlai/mturk): contains code for setting up Mechanical Turk, as well as sample MTurk tasks
90 - [**messenger**](./parlai/messenger): contains code for interfacing with Facebook Messenger
91 - [**zoo**](./parlai/zoo): contains code to directly download and use pretrained models from our model zoo
