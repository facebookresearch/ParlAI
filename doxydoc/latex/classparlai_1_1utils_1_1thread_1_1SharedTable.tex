\subsection{Detailed Description}
\begin{DoxyVerb}Provides a simple shared-memory table of integers, floats, or strings.

Use this class as follows:

.. code-block:: python

    tbl = SharedTable({'cnt': 0})
    with tbl.get_lock():
        tbl['startTime'] = time.time()
    for i in range(10):
        with tbl.get_lock():
            tbl['cnt'] += 1
\end{DoxyVerb}
