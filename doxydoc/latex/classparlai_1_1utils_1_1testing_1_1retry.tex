\subsection{Detailed Description}
\begin{DoxyVerb}Decorator for flaky tests. Test is run up to ntries times, retrying on failure.

On the last time, the test will simply fail.

>>> @retry(ntries=10)
... def test_flaky(self):
...     import random
...     self.assertLess(0.5, random.random())
\end{DoxyVerb}
